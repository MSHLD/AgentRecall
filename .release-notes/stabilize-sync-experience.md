# 会话与 Skills 同步更清晰、更可靠

## 新增功能

- ☁️ 会话同步现在会并排展示本地与云端的更新时间和同步状态，支持批量上传、删除云端副本，并在双方都发生变化时明确提示冲突。
- 📦 Skills 现在使用统一清单管理本地与云端副本，可在详情中分别查看本地内容、云端版本和文件差异，也支持批量上传、下载或删除云端 Skill。
- 🧹 新增 `agent-session-search uninstall`，可清理应用写入的集成配置和缓存，同时保留会话数据库、同步配置与用户偏好。
- 🧭 首次配置或同步结构需要更新时，可直接复制最新 SQL 并打开对应项目的数据库编辑页面，执行后原地刷新即可继续。

## Bug 修复

- 修复重复上传同一会话仍被误判为内容变化的问题。
- 修复同名 Skill 可能相互覆盖、大型 Skill 上传失败后误删其他版本文件，以及应用更新失败后旧版本无法继续启动的问题。
- 修复远程 Skill 的状态标签过多时名称显示不清，以及远程同步检查误报授权或错误引导执行 SQL 的问题。
- 修复关闭云端会话预览后直接退出同步清单，以及 Skills 上传完成后仍停留在“正在上传”状态的问题。
